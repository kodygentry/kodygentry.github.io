# Released Games Assets

This folder contains cover images/GIFs for your released games on itch.io.

## Files to Add:

### Midnight Slice Demo
- **Filename**: `midnight-slice.gif`
- **Description**: Animated GIF or static image for Midnight Slice Demo
- **Recommended size**: 800x600 or similar aspect ratio

### Eternal Burn
- **Filename**: `eternal-burn.gif`
- **Description**: Animated GIF or static image for Eternal Burn
- **Recommended size**: 800x600 or similar aspect ratio

## How to Get Your Game Cover Images from itch.io:

1. Go to your game's page on itch.io
2. Right-click on the cover image/GIF
3. Select "Save image as..." or "Copy image address"
4. Save it to this folder with the correct filename
5. Refresh your portfolio page

## Notes:
- GIF format is preferred for animated covers
- PNG/JPG also work for static images
- Keep file sizes under 5MB for optimal loading
